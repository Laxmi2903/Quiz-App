# Quiz-App
Project Title: Full-Stack Quiz Application Overview: This application is a full-stack platform for interactive quiz-taking, built with React for the frontend, Spring Boot for the backend API, and MySQL for data management. It provides a seamless experience for users to view, take, and track quiz scores. Key Technologies: Frontend: React.js, Axios (for API requests), CSS. Backend: Java Spring Boot, RESTful APIs, Spring Data JPA. Database: MySQL. Setup & Execution: Ensure MySQL is running with the required schema. Start the backend (QuizBackendApplication) on port 8080. Navigate to the frontend directory and run npm run dev (port 5173). Access the quiz interface via http://localhost:5173. This project demonstrates my ability to integrate frontend interfaces with backend services to build a functional, end-to-end web application.
